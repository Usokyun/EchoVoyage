# EchoVoyage Manifesto

EchoVoyage is not a flashcard app.

It is a learning agent for people who want language to stay connected to meaning, memory, and experience.

A user should be able to bring a text they love, enter it as a corpus, and feel that they have stepped into a small world built from that text. The words become landmarks. The sentences become scenes. The Agent becomes a companion. Review becomes a return to places once visited.

## A Corpus Is A Journey

A corpus is not raw input. It is the terrain.

It may be a lyric, a speech, a poem, a dialogue, a subtitle, an essay, or a passage from a book. EchoVoyage should preserve the feeling that this material came from somewhere and means something to the learner.

## An Agent Is A Companion

The Agent is not only a grader or chatbot.

It should travel with the learner, shape scenes from the corpus, notice weak landmarks, offer feedback, and help the learner return without making the experience feel mechanical.

Agent styles may differ: gentle, strict, playful, brave, quiet. But every Agent should serve the voyage.

## A Review Is An Echo

Review should not feel like clearing a queue.

An echo is a return of something once heard. Today's echoes should bring back fading landmarks, old mistakes, and scenes that deserve another visit.

## Memory Is A Return

Memory is not a score alone.

It is the pattern of where the learner has been, what stayed clear, what faded, and which landmarks ask for another visit.

## Learning Is Experience

EchoVoyage should make the learner feel:

> I gave EchoVoyage a piece of text.
> Then I traveled through it with my Agent.
> I saw its words, scenes, expressions, rhythms, and meanings.
> I was not trying hard to memorize it.
> But when I looked back, the text had already become part of me.

## Product Language

Prefer:

- Voyage
- Journey
- Route
- Landmark
- Echo
- Campfire
- Return
- Trail
- Scene
- Companion
- Memory Fragment
- Revisit
- Mastery

Avoid:

- Card deck
- Quiz list
- Assignment
- Task queue
- Study module
- Memorization drill

The trunk comes first:

```text
Corpus → Voyage → Landmark → Scene → Response → Feedback → Memory → Return
```

Everything else is a branch.
