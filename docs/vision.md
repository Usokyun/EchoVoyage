# Vision

EchoVoyage is an open-source learning agent that turns meaningful language material into a journey.

The product should feel romantic, personal, adventurous, and memorable. It should help a learner travel through a text they care about instead of forcing them through a generic drill flow.

## Product Feeling

EchoVoyage should feel like:

- A language journey
- A memory adventure
- A personalized companion
- A return to meaningful text
- A world built from the user's own corpus

It should not feel like:

- A traditional Anki clone
- A vocabulary drilling tool
- A generic AI chatbot
- A boring course generator
- A productivity dashboard

## Who It Is For

EchoVoyage is for learners who bring their own material:

- A lyric they love
- A speech they want to understand deeply
- A dialogue from a show or game
- A poem, essay, subtitle, or passage
- Any text that has emotional or intellectual weight

The learner should feel that the system respects the source material instead of reducing it into disconnected items.

## Why Meaningful Corpora Matter

Language memory is stronger when it is tied to scenes, voices, rhythms, and personal interest.

EchoVoyage begins from the user's corpus because a meaningful text already contains motivation. The system's job is to reveal landmarks inside it, not replace it with generic lessons.

## First Voyage Feeling

After one voyage, the user should be able to say:

> I gave EchoVoyage a piece of text.
> Then I traveled through it with my Agent.
> I saw its words, scenes, expressions, rhythms, and meanings.
> I was not trying hard to memorize it.
> But when I looked back, the text had already become part of me.

## Design North Star

Build the trunk first:

```text
Corpus → Voyage → Landmark → Scene → Response → Feedback → Memory → Return
```

A small but memorable trunk is better than a large system with no feeling.
